# FPGA Projects
A collection of FPGA projects written in VHDL and System Verilog. The projects utilized several PMODs interfaced with the Zybo Z7-10 development board. The projects were developed in Xilinx Vivado.

## Projects
* [Ultrasonic Sensor Controller and SSD](https://github.com/michaelgranberryii/fpga-ultrasonic-sensor)
* [Guessing Game](https://github.com/michaelgranberryii/fpga-guessing-game)
* [Simon Game](https://github.com/michaelgranberryii/fpga-simon-game)
* [Digital Lock FSM](https://github.com/michaelgranberryii/fpga-fsm-digital-lock)
* [PWM Controller: RGB LEDs and Servo Motor](https://github.com/michaelgranberryii/fpga-pwm)
* [UART Controller and LEDs](https://github.com/michaelgranberryii/fpga-uart)
* [VGA Controller](https://github.com/michaelgranberryii/fpga-vga)
